# dory
Quick Guide and Commands in the Terminal

## Set Alias
Copy *".aliases_dory"* on your user path.
For Example *cp .aliases_dory ~/*

## Add your Alias in the Shell
>- Example [.bashrc]
>>vim ~/.bashrc

>- Example [.zshrc]
>>vim ~/.zshrc

>- Paste the line to link the alias
>>source ~/.aliases_dory

> Save the file

## Update your Shell

>>source ~/.bashrc

> Example [.zshrc]
